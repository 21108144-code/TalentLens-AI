# TalentLens AI - Notebooks

This directory contains Jupyter notebooks for experimentation and EDA.

## Notebooks

1. **01_data_exploration.ipynb** - Dataset exploration and analysis
2. **02_nlp_experiments.ipynb** - NLP preprocessing experiments
3. **03_model_training.ipynb** - Model training and comparison
4. **04_evaluation.ipynb** - Model evaluation and metrics

## Running Notebooks

```bash
# Install Jupyter
pip install jupyter

# Start Jupyter server
jupyter notebook

# Or use JupyterLab
pip install jupyterlab
jupyter lab
```

## Google Colab

These notebooks can also be run in Google Colab for free GPU access:
1. Upload the notebook to Google Drive
2. Open with Google Colab
3. Enable GPU: Runtime > Change runtime type > GPU
